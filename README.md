# React + Sockets: Aplicaciones en tiempo real con Bun

Bitácora de aprendizaje sobre el desarrollo de aplicaciones en tiempo real usando WebSockets nativos en Bun junto con React. Aquí guardo mi progreso, notas y ejercicios del curso.

## ¿Qué hay en este repo?
- Notas sobre la implementación nativa de WebSockets en Bun.
- Ejercicios prácticos de comunicación bidireccional.
- Integración de sockets con el estado de React.
- Documentación del paso a paso.
